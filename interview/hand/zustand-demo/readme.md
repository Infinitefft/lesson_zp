## 开始实现
- 实现一个最简单的 Store
  存储以及管理状态，怎么存，怎么取，怎么改
- 加入**订阅机制**
  抖音的网红（store 发布者）和订阅者（订阅关系）
  以后发布者发布内容（状态改变）
  订阅者收到消息，并执行
  - store 对象上添加subscribe 方法
    接收一个函数
  - state 改变的时候，执行这个函数
