## HTML 标签分为两类
    - 块级元素 block(div等) 独占一行  可以设置宽度高度，负责布局，结构
      - 宽度默认整个一行
      - 高低如果没有设定由内容撑开
      - h3 文字高度 行高 情书的行高
    - 行内元素 inline(span) 不独占一行 不能设置宽度高度
